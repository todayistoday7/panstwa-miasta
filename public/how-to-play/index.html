How to Play Countries & Cities Online

Countries & Cities (known in Polish as Państwa-Miasta) is a classic multiplayer word game you can now play online with friends, family or colleagues — free, no registration needed, from any phone or computer.

The Basic Rules

The game is played in rounds. At the start of each round, a random letter is drawn. All players simultaneously fill in categories — writing words that start with that letter. The first player to complete all their categories shouts STOP! — everyone else must stop writing.

1. Create or Join a Room

The host creates a room and gets a 5-letter code. They share it with others — over text, WhatsApp, or out loud. Other players type the code to join. You need at least 2 players, up to 12.

2. Drawing a Letter

The host draws a letter by pressing the button. Each letter can only appear once per game — no repeats. Difficult letters like X and Q may be excluded depending on the language setting.

3. Filling in Categories

Once the letter is drawn, everyone writes simultaneously. Default categories include: Country, City, Animal, Plant, Thing, Name, Colour, Job. The host can customise categories before the game starts.

4. STOP! — Ending the Round

The first player to fill in all categories hits STOP. Remaining players get a short grace period to finish (configurable, default 20 seconds). After that, answers are revealed.

💡 Pro tip: Go for niche, unusual answers. "El Salvador" beats "Spain" for S every time — the less likely someone else thinks of it, the more points you score.

The Challenge System

After each round, players can challenge any answer they think is invalid by clicking the ⚠ Challenge button. All players then vote whether the word is acceptable. The majority decides.

  • If the word is rejected — the player who wrote it scores 0 for that category
  • If the word is accepted — points stand and the challenge fails
  • The challenged player can argue their case before the vote

Rounds and Endgame

Games last 5 rounds by default. The host can set 3 to 10 rounds before starting. After all rounds, a final leaderboard shows the total score for every player.

Host Settings

  • Number of rounds — 3 to 10
  • Grace period after STOP — 10, 20, or 30 seconds
  • Categories — choose from presets or customise
  • Language — PL, EN, DE, FR, ES
  • Room visibility — private (code only) or public (listed on Game Hub)

Multilingual Play

The game runs in five languages: Polish, English, German, French, and Spanish. The language setting changes category names and the interface. Players speaking different languages can share one room.

FAQ

Is it free?

Yes, completely. No sign-up, no fees, no ads.

How many players can join?

2 to 12 players per room.

Does it work on mobile?

Yes — it runs in any browser on any device.

What if a player disconnects?

Players can rejoin within 30 minutes using the same room code and name.

How long is the room code valid?

Room codes stay active for 24 hours before the game starts. Once the game ends, the room expires after 1 hour.
